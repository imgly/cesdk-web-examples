# URI Resolver Guide (Browser)

This example demonstrates how to customize URI resolution in CE.SDK for the browser.

## Features

- Understanding default URI resolution behavior
- Testing resolution with `getAbsoluteURI()`
- Implementing custom URI resolvers
- Adding authentication tokens to asset URIs
- Path transformation for different environments
- Security policy enforcement
- Removing custom resolvers

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
