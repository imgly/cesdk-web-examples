# Placeholders - Browser Guide

This example demonstrates how to use placeholders in CE.SDK to mark editable areas within templates.

## Features

- Checking placeholder support
- Enabling placeholder behavior
- Configuring placeholder controls (overlay and button)
- Managing placeholder interaction modes
- Combining placeholders with other template constraints

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
