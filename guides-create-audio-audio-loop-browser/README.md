# Loop Audio - CE.SDK Browser Guide

This example demonstrates how to use audio looping in CE.SDK.

## Features

- Enable and disable audio looping
- Control playback duration with looping
- Loop trimmed audio segments
- Query looping state

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
