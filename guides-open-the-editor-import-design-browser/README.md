# Import Design - Browser Example

This example demonstrates how to import designs into CE.SDK from various sources:

- Load previously saved CE.SDK scenes from URLs or strings
- Load archived scenes (.zip files) with bundled assets
- Import from professional design tools (Photoshop, InDesign)
- Create editable scenes from images and videos

## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features Demonstrated

1. **Load from URL** - Load scene files from remote URLs
2. **Load from String** - Restore scenes from string content
3. **Load from Archive** - Load self-contained archive files
4. **Create from Image** - Build editable scenes from images
5. **Create from Video** - Set up video editing scenes
6. **Modify Loaded Scenes** - Edit scenes immediately after loading
