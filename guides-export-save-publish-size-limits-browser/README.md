# Size Limits Guide - Browser

This example demonstrates how to configure and manage image and video size limits in CE.SDK.

## What This Example Covers

- Reading the current `maxImageSize` setting
- Configuring `maxImageSize` for different use cases
- Querying GPU texture size capabilities
- Validating export dimensions before exporting
- Handling size-related export errors

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Documentation

For complete documentation, see the [Size Limits Guide](https://img.ly/docs/cesdk/guides/export-save-publish/export/size-limits).
