# To Base64 - Browser Guide

This example demonstrates how to convert CE.SDK exports to Base64-encoded strings for embedding in URLs, storing in databases, or transmitting via APIs.

## Quick Start

```bash
npm install
npm run dev
```

## Features Demonstrated

- Exporting blocks to Blob using `engine.block.export()`
- Converting Blob to Base64 data URI using FileReader
- Working with different MIME types (PNG, JPEG, WebP)
- Using Base64 data URIs in HTML elements
