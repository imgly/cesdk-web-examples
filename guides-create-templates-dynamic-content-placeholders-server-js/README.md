# Placeholders - Server Guide (Node.js)

This example demonstrates how to use placeholders in CE.SDK for headless template creation and automation.

## Features

- Checking placeholder support programmatically
- Enabling placeholder behavior on graphic blocks
- Configuring scope-based placeholder permissions
- Batch processing multiple placeholders
- Exporting templates with placeholder configurations

## Getting Started

```bash
# Install dependencies
npm install

# Run the example
npm run dev

# Run all checks (TypeScript, ESLint, Prettier)
npm run check:all
```

## Output

The example generates:
- `output/placeholders-template.scene` - Reusable template with placeholder configurations
- `output/placeholders-result.png` - Visual preview of the configured template
