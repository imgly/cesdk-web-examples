# Unsplash Custom Asset Source - CE.SDK Browser Guide

This example demonstrates how to integrate custom asset sources into CE.SDK using Unsplash as an example.

## Features

- Custom asset source implementation
- Unsplash API integration
- Search and pagination support
- Asset translation and mapping
- Credits and licensing
- Local asset sources

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
