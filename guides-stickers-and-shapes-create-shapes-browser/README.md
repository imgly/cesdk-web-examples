# Create Shapes - CE.SDK Browser Guide

This example demonstrates how to programmatically create and configure shapes in CE.SDK.

## Features Demonstrated

- Creating graphic blocks
- Applying different shape types (rectangle, ellipse, star, polygon, line, vector path)
- Configuring shape-specific properties
- Combining shapes with fills
- Shape and fill independence

## Running the Example

```bash
npm install
npm run dev
```

## Building for Production

```bash
npm run build
```

## Capturing Hero Image

```bash
npm run capture-hero
```
