# Create Shapes - CE.SDK Server Guide (Node.js)

This example demonstrates how to programmatically create and configure shapes in CE.SDK for server-side Node.js applications.

## Features Demonstrated

- Creating graphic blocks headlessly
- Applying different shape types (rectangle, ellipse, star, polygon, line, vector path)
- Configuring shape-specific properties
- Combining shapes with fills
- Exporting scenes with shapes to image files

## Running the Example

```bash
npm install
npm run dev
```

The example will create a PNG file in the `output/` directory showing all created shapes.

## Building

```bash
npm run build
```
