# Content Moderation Browser Guide

This example demonstrates how to implement automated content moderation in CE.SDK to detect and flag potentially inappropriate images in designs.

## Features

- Find all image blocks in a design
- Extract image URLs from blocks
- Integrate with external moderation APIs
- Display validation results with severity levels
- Interactive block selection from results

## Setup

```bash
npm install
npm run dev
```

For the complete guide, see: [Moderate Content](https://img.ly/docs/cesdk/web/guides/rules/moderate-content/)
