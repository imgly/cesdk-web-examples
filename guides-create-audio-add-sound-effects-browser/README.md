# Add Sound Effects - CE.SDK Browser Guide

This example demonstrates how to add custom sound effects to audio compositions using audio buffers in CE.SDK.

## Features

- Creating audio buffer resources
- Loading audio files with Web Audio API
- Creating audio blocks from buffers
- Positioning sound effects on the timeline

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
