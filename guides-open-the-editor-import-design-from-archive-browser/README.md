# Import Design from Archive - CE.SDK Guide

This example demonstrates how to load self-contained CE.SDK archive files using `engine.scene.loadFromArchiveURL()`.

## What You'll Learn

- Load archived scenes with bundled assets
- Create object URLs for local archive files
- Handle user-uploaded archive files
- Understand archive contents and structure
- Modify loaded archive scenes

## Quick Start

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```
