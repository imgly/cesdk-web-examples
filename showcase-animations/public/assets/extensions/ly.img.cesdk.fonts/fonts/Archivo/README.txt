Archivo Variable Font
=====================

This download contains Archivo as both variable fonts and static fonts.

Archivo is a variable font with these axes:
  wdth
  wght

This means all the styles are contained in these files:
  Archivo-VariableFont_wdth,wght.ttf
  Archivo-Italic-VariableFont_wdth,wght.ttf

Get started
-----------

1. Install the font files you want to use

2. Use your app's font picker to view the font family and all the
available styles
